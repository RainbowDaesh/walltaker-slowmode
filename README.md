# Slowmode for Walltaker
Modifies your Walltaker link so your walltaker cannot be changed for one minute.

---
## Installation
1. Edit your link
2. Paste the contents of `wtslo.html` (or the html below) somewhere into your description
```html
<style>.modal .secondary{display:none}.modal h4,.modal p{visibility:hidden;position:relative}.modal h4::before,.modal p::before{content:"Whoa, easy, tiger! üêØ";visibility:visible;position:absolute}.modal p::before{content:"This wallpaper was just changed less than a minute ago. The owner of the link has set it so wallpapers cannot be changed so soon."}</style>
```
3. Save
